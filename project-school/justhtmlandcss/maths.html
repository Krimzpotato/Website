Mathematics Notes
We will be providing some maths knowledge here.

1. Algebra

Basic Algebraic Expressions
Algebra involves variables, constants, and operators. An algebraic expression can be a combination of these elements.
  • Example: 3x + 5 = 11
  • Here, 3 is the coefficient, x is the variable, and 5 is the constant.
Solving Equations
To solve an equation, we isolate the variable by applying inverse operations.
  • Example: 2x + 3 = 11
  • Solution: Subtract 3 from both sides → 2x = 8 → Divide by 2 → x = 4

2. Geometry

Area Of Basic Shapes
  • Rectangle: Area = Length × Width
  • Circle: Area = π × Radius²
Volume Of Solids
The volume of a solid is the amount of space it occupies. Common formulas include:
  • Cube: Volume = Side³
  • Cylinder: Volume = π × Radius² × Height

3. Calculus

Limits
The limit of a function describes its behavior as the input approaches a specific value.
  • Example: limx→0 (sin x)/x = 1
Derivatives
  • Example: The derivative of f(x) = x² is f'(x) = 2x

4. Trigonometry

Basic Trigonometric functions
  • sin(θ) = opposite/hypotenuse
  • cos(θ) = adjacent/hypotenuse
  • tan(θ) = opposite/adjacent
Trigonometric identities
Some useful trigonometric identities include:
  • sin²(θ) + cos²(θ) = 1
  • tan(θ) = sin(θ) / cos(θ)
